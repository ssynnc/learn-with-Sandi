# LearnWithSandi

LearnWithSandi is a modern language learning platform designed to help users learn Korean through structured video lessons and courses.  
The platform provides an easy and interactive way for students to improve their language skills through organized content and a clean learning interface.

---

## Features

- Video-based Korean language courses
- Beginner to advanced learning levels
- Clean and modern UI
- User authentication (Login / Sign Up)
- Course browsing system
- Student dashboard
- Responsive design (Desktop & Mobile)
- Blog and learning resources

---

## Tech Stack

This project is built using modern web development technologies:

- Next.js
- React
- TypeScript
- Tailwind CSS
- Node.js

## Installation

Follow these steps to run the project locally.

### 1. Clone the repository
git clone https://github.com/ssynnc/learn-with-Sandi.git

### 2. Navigate to the project folder
cd learn-with-Sandi

### 3. Install dependencies
npm install

### 4. Run the development server
npm run dev

### 5. Open in browser
http://localhost:3000
